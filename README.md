# Управление карточками доступа

Веб-приложение для управления пользователями и карточками доступа с историей изменений и печатью.

## Возможности

- CRUD пользователей и копирование карточек пользователей
- Справочники должностей, отделов, ресурсов и требований к ПО
- CRUD карточек доступа с множественным выбором ресурсов
- История изменений карточек и просмотр/печать карточек
- Фильтрация пользователей по статусу

## Запуск

```bash
docker-compose up --build
```

После запуска откройте: `http://localhost:8080`.

## Структура

- `public/` — фронтенд и точка входа PHP
- `src/` — модульные контроллеры и роутер
- `db/init.sql` — схема и тестовые данные
- `docker-compose.yml`/`Dockerfile` — деплой в Docker
